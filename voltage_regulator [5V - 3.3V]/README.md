# Buzzer PCB Design

## 📌 Overview
A custom PCB designed to drive a buzzer circuit with
controlled switching and onboard power regulation.

## ⚡ Key Specifications
- Main Component: Buzzer
- Layers: 2-layer PCB
- Designed using: KiCad

## 📂 Files Included
- `buzzer.kicad_pcb` — PCB layout file
- `buzzer.kicad_sch` — Schematic file
- `/gerber` — Manufacturing ready Gerber files

## 🛠️ Tools Used
- KiCad 7.0

## 🏭 Manufacturing
Gerber files in `/gerber` folder are ready to order
from JLCPCB or PCBWay.
